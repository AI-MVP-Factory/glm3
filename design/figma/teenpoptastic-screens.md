# TeenPoptastic MVP - Figma Screen Documentation

## Overview
TeenPoptastic is a social discovery app designed specifically for teenagers to find like-minded peers based on shared interests and vibes. This document outlines all screens with teen-focused emotional design elements.

---

## Emotional Design Philosophy
- **Authenticity:** Real connections over curated perfection
- **Safety:** Built-in protections for teen users
- **Expressiveness:** Multiple ways to show personality
- **Trendiness:** Current design aesthetics teens love
- **Playfulness:** Gamified elements for engagement
- **Inclusivity:** Diverse representation in all designs

---

## 1. Onboarding Flow

### Screen 1: Vibe Check
**Purpose:** Understanding user's interests and personality
**Layout:**
- Full-screen gradient background (purple to pink)
- "What's Your Vibe?" headline with playful typography
- Interest selection interface:
  - Grid of interest cards with icons and names
  - Categories: Music, Sports, Gaming, Art, Fashion, Food, Movies, School
  - Multi-select with visual feedback (3+ selections required)
  - "Your Interests" progress bar
- Next button activates when minimum selections made

**Emotional Design Notes:**
- Bright, energetic colors that feel youthful
- Smooth animations when selecting interests
- Progress indicator to reduce decision fatigue
- Option to add custom interests for personalization
- Friendly mascot character giving thumbs up

### Screen 2: Permission & Safety
**Purpose:** Establish trust and safety protocols
**Layout:**
- Clean, trustworthy design with teal accent
- "Stay Safe, Have Fun" headline
- Safety features checklist:
  - Age verification toggle
  - Parental consent info
  - Privacy controls explanation
  - Reporting system overview
  - Content guidelines summary
- "I Understand" checkbox and Continue button
- Support contact information at bottom

**Emotional Design Notes:**
- Reassuring tone without being patronizing
- Visual shields and lock icons for security
- Clear, concise language teens understand
- Empowering messaging about taking control
- Professional yet approachable aesthetic

### Screen 3: Profile Setup
**Purpose:** Create authentic teen profile
**Layout:**
- Gradient background matching brand
- Profile photo upload with AR effects option
- Username creation with availability check
- Bio text area with character counter
- Vibe tags selection (up to 10)
- Privacy settings:
  - Profile visibility (Everyone/Match only/Hidden)
  - Show age toggle
  - Share location toggle
- "Complete Profile" CTA

**Emotional Design Notes:**
- Fun profile photo effects and filters
- Username suggestions when taken
- Bio character counter with playful limits
- Vibe tags with emoji icons for quick selection
- Progressive disclosure for complex options
- Celebratory confetti when complete

## 2. Vibe Matching

### Screen 1: Card Stack
**Purpose:** Discover potential matches through gamified interface
**Layout:**
- Full-screen card stack interface
- Profile card design:
  - Profile photo with name and age
  - 3-4 main interest tags with icons
  - Shared interests highlighted
  - Quick stats (distance, % match)
- Action buttons:
  - X (dislike) - bottom left
  - Favorite (super like) - bottom center
  - Check (like) - bottom right
- Swipe indicators showing cards remaining
- Bottom navigation hint

**Emotional Design Notes:**
- Tactile feedback on swipe gestures
- Smooth card animations and transitions
- Match percentage creates excitement
- "Spark" animation when interests align
- Sound effects for actions (subtle)
- Visual loading states between cards

### Screen 2: Match Reveal
**Purpose:** Celebrate mutual connections and initiate conversation
**Layout:**
- Celebration animation (confetti, sparkles)
- "It's a Match!" headline with emoji
- Both profile cards side by side
- Shared interests highlighted with connecting lines
- "Match Score" percentage prominently displayed
- Action buttons:
  - Start Chat
  - View Profile
  - Keep Swiping
- Match statistics (time, distance)

**Emotional Design Notes:**
- Excitement through animation and sound
- Shared interests create instant connection points
- Clear CTA to move relationship forward
- Celebration feels rewarding and special
- Match score validates connection quality
- Smooth transition to chat interface

## Teen-Focused Design Specifications

### Color Palette
- Primary: #7C3AED (Purple) - Youthful and energetic
- Secondary: #EC4899 (Pink) - Fun and social
- Accent: #06B6D4 (Teal) - Trust and safety
- Success: #10B981 (Green) - Positive reinforcement
- Warning: #F59E0B (Amber) - Attention needed
- Danger: #EF4444 (Red) - Stop/Alert
- Background: #FDF4FF (Ultra light purple)
- Text: #1F2937 (Dark gray)

### Typography
- Headings: Poppins, Bold, 28px
- Subheadings: Poppins, SemiBold, 20px
- Body: Inter, Regular, 16px
- Labels: Inter, Medium, 14px
- Buttons: Poppins, SemiBold, 16px

### Spacing
- Unit: 8px (base)
- Padding: 16px, 20px, 24px
- Margin: 8px, 12px, 16px
- Component spacing: 16px
- Card elevation: 6px with soft shadow

### Iconography
- Style: Outlined with playful variations
- Size: 24px (standard), 28px (large)
- Colors: Primary brand, accent for actions
- States: Clear hover and active states
- Custom illustrations for brand personality

### Animation Language
- Bouncy: 400ms cubic-bezier(0.68, -0.55, 0.265, 1.55)
- Smooth: 300ms cubic-bezier(0.4, 0, 0.2, 1)
- Micro-interactions on all buttons
- Swipe gestures with haptic feedback
- Loading states with skeleton screens
- Success celebrations with particle effects

### Teen Psychology Integration
1. **FOMO Elements:** Limited-time stories, exclusive features
2. **Social Proof:** Match counts, activity streaks
3. **Self-Expression:** Multiple customization options
4. **Recognition:** Achievement badges and levels
5. **Belonging:** Group interests and communities
6. **Control:** Privacy settings and safety tools

### Accessibility Guidelines
- All colors meet WCAG AA contrast
- Minimum 44px touch targets
- Clear focus indicators
- Screen reader friendly
- Dynamic text scaling
- Reduced motion options

### Emotional Design Patterns
- **Excitement:** Bright colors, celebrations, rewards
- **Trust:** Clear safety features, transparency
- **Belonging:** Community features, shared interests
- **Expression:** Customization, multiple media types
- **Security:** Privacy controls, age verification
- **Playfulness:** Gamification, animations, interactions

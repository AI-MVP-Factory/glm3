# Screenshot Template Guidelines - Quick Reference

## Device Dimensions

### Mobile
- **iPhone 15/14 Pro**: 1290×2796 (19.5:9)
- **iPad Pro 13"**: 2048×2732 (4:3)

### Desktop
- **MacBook 14"**: 1512×982
- **MacBook 16"**: 1728×1117
- **iMac 24"**: 1080×1920

## Text Overlay Specifications

| Element | Font | Size | Weight | Color |
|---------|------|------|--------|-------|
| **Headline** | SF Pro Display | 28-32pt | Bold | White/Black |
| **Subheading** | SF Pro Text | 16-18pt | Regular/Medium | Muted |
| **Callout** | SF Pro Display | 14-16pt | Bold | High Contrast |
| **Badge** | SF Pro Display | 12-14pt | Bold | White + Colored BG |

## Naming Convention
`[app]-[screen]-[device]-[locale].png`

Example: `taskflow-home-iphone15-en.png`

## File Structure
```
/design/screenshots/
├── taskflow/
├── teenpoptastic/
└── memecraftvibe/
```

## Figma Components Needed
- Device frames (all sizes)
- Text styles (headline, subheading, callout, badge)
- Color palettes per app
- Badge component variants

## Quick Checklist
□ Correct device frame
□ High-quality PNG export
□ Follows naming convention
□ Text is readable
□ Brand colors applied
□ Appropriate for target audience
